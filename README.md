# Face Generator

Python script containing TensorFlow DCGAN implementation. It was trained on a [Celebrities](https://www.kaggle.com/greg115/celebrities-100k) dataset.

Corresponding Medium article: [Face Generator - Generating Artificial Faces with Machine Learning ðŸ§‘](https://towardsdatascience.com/face-generator-generating-artificial-faces-with-machine-learning-9e8c3d6c1ead).

## DCGAN
Network architecture by [Radford et al., 2015](https://arxiv.org/abs/1511.06434).
